  3. 插件动态广播执行入口hook
    sendDynamicBroadcast函数通过一系列的反射，最终拿到了插件receiver的远程binderProx，用于执行binderProx上的performReceive方法。这个点，刚好是FrameWork中调度receiver起始点。 到这一步，动态注册的receiver也处理完毕。
